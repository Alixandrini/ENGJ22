Tabalho de pesquisa para continuação do estudo de aplicação de ML no ajustamento de observações geodésicas.
Os alunos vão comparar o método de regressão linear (ajustamento) com regressões ML da bibliotéca scikitlearn como: SVC, RFC, RLC ...

O problema consiste no ajustamento de um geoide local baseado na observação de valores de altitude elipsoidal na rede altimétrica de primeira ordem em Salvador.
